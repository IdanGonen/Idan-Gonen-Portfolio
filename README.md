# Idan Gonen - Portfolio

A modern, responsive personal portfolio built with React, TypeScript, and Tailwind CSS.

## 🌟 Live Demo

[https://IdanGonen.github.io/Idan-Gonen-Portfolio/](https://IdanGonen.github.io/Idan-Gonen-Portfolio/)

## 🚀 Quick Start

```bash
npm install
npm run dev
npm run deploy
```

## 🛠️ Tech Stack

React 19 • TypeScript • Tailwind CSS • Vite • React Router
